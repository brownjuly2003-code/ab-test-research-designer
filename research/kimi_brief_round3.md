# Research brief: лучшие мини-презентации продуктов (30-90 секунд)

## Контекст

Пользователь готовит видео-питч open-source инструмента **AB_TEST** (статистический движок для A/B-экспериментов: student-t, sequential checks, прозрачный механизм без peeking-багов). Целевая аудитория — нанимающие менеджеры в data engineering / analytics / experimentation. Длительности: 30 / 60 / 90 секунд.

## Задача

Найди **10-15 лучших образцов** мини-презентаций продуктов с **рабочими URL**, которые можно открыть и посмотреть прямо сейчас.

### Критерии «лучших»

- **Длительность**: 20-120 секунд (диапазон около цели — 30/60/90).
- **Жанр приоритетный**: dev tools / open-source / data products / SaaS / B2B инфраструктура. Меньше — но допустимо: consumer SaaS, AI продукты.
- **Качество**: видео цитируется в продуктовых сообществах, набрало значимое внимание (X / HN / YouTube views), либо является эталоном жанра.
- **Релевантность для AB_TEST**: упор на demo-walkthrough, technical credibility, понятное «что это и зачем» за минуту.

### Что нужно по каждому образцу

```
### {{Название продукта}} — {{Жанр}} ({{длительность}})
- **URL**: прямая ссылка на видео (YouTube / Vimeo / встроенный плеер на лендинге / Twitter)
- **Компания / автор**: ...
- **Год**: ...
- **Длительность**: ровно сколько секунд
- **Жанр**: launch trailer / explainer / demo walkthrough / founder pitch / feature reel
- **Почему работает**: 2-3 пункта про конкретные приёмы (hook за первые 3 секунды, on-screen text, темп монтажа, голос за кадром vs talking head, screen recording с zoom-in, etc.)
- **Что можно унести в AB_TEST**: 1-2 идеи, как этот приём адаптировать под dev-tool про статистику
```

### Жанровые категории, которые желательно покрыть

1. **Fireship-style 100-second explainers** (если есть свежие dev-tool примеры — Linear, Vercel, Supabase, Posthog, etc.)
2. **Лендинговые hero-видео** dev-tool компаний (Statsig, GrowthBook, Posthog, Plausible, Linear, Vercel, Supabase, Resend, Cal.com, Tinybird) — что играет в hero-блоке их главной страницы
3. **YC Demo Day pitches** (3 мин, но первые 30-90с — образцовый hook)
4. **Show HN / Product Hunt** видео-демонстрации, набравшие традицию (top 10 по жанру за 2023-2025)
5. **Founder-driven micro-demos** (Theo, Pieter Levels, Guillermo Rauch, Tobi Lütke стиль) на X / YouTube
6. **Технические launch-видео** (Bun, Deno, htmx, tRPC, Drizzle ORM — короткие официальные tease/launch видео)

### Что НЕ нужно

- Корпоративный glossy с актёрами и stock-музыкой (Salesforce / Oracle стиль) — нерелевантно.
- Длинные конференционные доклады (>3 мин).
- Видео без рабочего URL.
- TikTok / Reels без прямой ссылки на продукт (не воспроизводимо).

### Особый бонус — анти-примеры

**3-5 коротких видео, которые НЕ работают**, с разбором почему (плохой hook, генерик «введите свой email», слайды-перегрузка, нет демо). С URL.
