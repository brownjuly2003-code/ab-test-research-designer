## Notes
- **Не** добавлять новые npm-deps. `vitest-axe` уже доступен через зависимость; если нет — использовать `axe-core` напрямую (`import { axe } from "axe-core"` + `await axe.run(container)`).
- **Не** править контрастность цветов без реального axe-сигнала — WCAG AA = 4.5:1 для текста, 3:1 для UI; если axe не жалуется — не трогать.
- **Не** трогать `color-contrast` в рекчартах/svg без явной violation.
- Если focus trap для `TemplateGallery`/`ShortcutHelp` требует новой библиотеки (например, `focus-trap-react`) — **не** ставить; реализовать ручками через `useEffect` + key-listener, иначе bundle budget может просесть.
- Если violation не воспроизводится в jsdom-окружении (например, layout-based) — пометить в отчёте как «jsdom-limited, проверено вручную в Playwright» и закрыть через e2e-smoke (в `app/frontend/src/test/e2e-smoke.spec.ts` добавить `expect(page).toHaveNoAxeViolations()` через `@axe-core/playwright` — **только если** уже стоит; не ставить новую deps).
- Recharts SVG часто тригерит `svg-img-alt` warning — задокументировать как «декоративная графика, данные дублированы в SensitivityTable», и исключить правило точечно, не глобально.
- Backend test_performance p95-guard может флапнуть — перезапустить один раз.
- **Не** пушить на remote.
